# Tubie-Man Sprites

The sprite sheet consists of multiple sprites which are 330 x 330 px, on a square grid.

If the dimensions and or number of sprites are changed in atlas.png, atlas.json must also be updated to accomodate the new sprite, and/or changed dimensions. Simply changing the graphics of the sprite sheet in situ does not require updates be made to atlas.json.

The atlas JSON file can be generated with tools such as TexturePacker, however no tools were used to generate the one used for this game.
